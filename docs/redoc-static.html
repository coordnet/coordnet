Query a buddy

Query a buddy using the given nodes and level. The level defines the depth of the query, level 0 meaning that only the node detail page of the node specified is queried, level 1 includes the specified node's graph (subnodes), level 2 will also add the subnodes detail pages to the context, etc.

Authorizations:
knoxApiTokencookieAuthNone
path Parameters
public_id
required
string <uuid>
Request Body schema:
required
message
string

Message sent to the buddy

nodes
required
Array of strings <uuid> [ items <uuid > ]

List of nodes to use as context

level
integer

Level of depth to query the nodes
